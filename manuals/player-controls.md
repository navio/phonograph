# Playback Controls

## Player Bar Essentials

The player bar provides quick access to:

- Play/pause
- Skip forward/back
- Current progress and duration
- Queue access

## Queue Behavior

Use the Playlist area to:

- Reorder upcoming episodes.
- Remove queued items.
- Start playback from any queued episode.

## Per-Podcast Preferences

Phonograph can remember settings by podcast, including:

- Preferred playback speed.
- Intro/outro skip timing when available.

These preferences are applied automatically when returning to the same show.

## Media Session Integration

On compatible platforms, Phonograph integrates with system media controls so you can:

- Play or pause from lock screen/headset controls.
- Resume without returning to the browser tab.

## Practical Tips

- Keep one episode active to avoid accidental queue resets.
- Check network status when long episodes stall.
- Use playlist order to stage multi-episode sessions.
