# Troubleshooting

## Audio Will Not Start

Try the following:

1. Confirm your device has network connectivity.
2. Refresh the page and retry playback.
3. Open another episode to isolate feed-specific issues.

## Player Looks Stuck

- Pause and play again.
- Skip to another queued episode.
- Reopen the app to reinitialize the player session.

## Discovery/Search Feels Empty

- Retry after a short delay.
- Verify ad blockers or strict privacy extensions are not blocking API calls.
- Confirm the issue persists in an incognito/private window.

## Library Changes Not Showing

- Refresh the app once to reload the latest local state.
- Confirm changes are in the same browser profile where they were made.

## PWA Install Option Missing

- Ensure HTTPS is enabled (production domain is HTTPS).
- Update to a browser with PWA install support.
- Open the site once and interact before checking install menu again.

## Reporting a Bug

When reporting issues, include:

- Device and browser version.
- The exact podcast or episode affected.
- A short sequence of reproduction steps.
- Timestamp and screenshot if available.
